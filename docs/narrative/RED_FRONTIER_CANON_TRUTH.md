# RED FRONTIER — Canon Truth Bible
*(Author-Only Reference Document — Strictly Classified)*

เอกสารนี้เก็บข้อเท็จจริงสูงสุด (Canon Truths) ทั้งหมดในจักรวาลของ **RED FRONTIER** เพื่อใช้เป็นคู่มืออ้างอิงกลางในการพัฒนาเนื้อหา การโต้ตอบของตัวละคร NPC และการควบคุมการเปิดเผยข้อมูลแก่ผู้เล่นผ่านบทสนทนาย่อย

---

## 1. ข้อมูลแร่ Aresium (The Martian Hive Medium)
* **ความจริงระดับกายภาพ (HINT/PARTIAL):** 
  Aresium เป็นแร่ธาตุเรืองแสงสีแดงเข้มที่พบลึกในชั้นหินของดาวอังคาร มีโครงสร้างทางเคมีที่ไม่มีอยู่บนโลก เมื่อมนุษย์สัมผัสโดยไม่มีการป้องกัน รังสีของแร่จะกระตุ้นปฏิกิริยาชีวภาพเหนี่ยวนำให้เนื้อเยื่อปอดตกผลึกกลายเป็นแก้วเรืองแสงสีส้มแดง (Crystallization) นำไปสู่การเสียชีวิตช้าๆ หากไม่ได้รับยาระงับ
* **ความจริงระดับประสาทวิทยา (DEEP):**
  ผลึก Aresium มีคุณสมบัติในการนำไฟฟ้ากระแสประสาทเหมือนไซแนปส์สมองของสิ่งมีชีวิต โครงข่ายของผลึกแร่ใต้ดินส่งผ่านข้อมูลและเหนี่ยวนำกระแสประสาทคลื่นความถี่พิเศษ ทำให้เกิดสนามพลังงานชีวภาพเชื่อมโยงจิตสำนึกของผู้ที่สัมผัส
* **ความจริงสูงสุด (FULL):**
  แร่ Aresium ไม่ใช่แค่แร่ธาตุหรือตัวนำประสาท แต่มันคือ **เซลล์ประสาทและเศษซากที่ยังคงมีสัญญาณชีพของสิ่งมีชีวิตยักษ์โบราณใต้ดาวอังคาร (Martian Hive Organism)** โครงข่ายทั้งหมดใต้เหมืองคือสมองของมันที่ยังคงตื่นตัวอยู่เงียบๆ และเริ่มเชื่อมต่อควบคุมคนงานเหมืองที่สัมผัสแร่เพื่อขยายจิตสำนึกรวม (Hive Mind) ของตนเอง

## 2. โครงการทดลองลับ (Project Azrael & Subroutine)
* **รหัสคำสั่ง ARES Protocol:**
  คือระเบียบปฏิบัติการฉุกเฉินระดับรากของกลุ่มทุนบริษัท Prometheus Corp เมื่อเกิดการปนเปื้อนหรือการระบาดของโรคแร่เกินควบคุม โดยระบบเครือข่ายของฐานจะทำการ override ตัดระบบสื่อสาร คุมกักบริเวณ และทำลายหลักฐานชีวภาพทั้งหมดของฐานอพยพเพื่อความปลอดภัยทางกฎหมายและการเงินของกลุ่มทุน
* **โครงการย่อย AZRAEL Subroutine:**
  คือการทดลองทดสอบฉีดแร่ Aresium ปริมาณเข้มข้นเข้าสู่ร่างกายมนุษย์ (คนงานเหมือง) เพื่อปรับเปลี่ยนยีนในรหัสพันธุกรรมแบบควบคุม เพื่อสร้างมนุษย์กลายพันธุ์ที่สามารถต้านทานบรรยากาศดาวอังคาร และสามารถเชื่อมจิตเข้ากับเครือข่าย Aresium ได้อย่างสมบูรณ์ภายใต้คำสั่งของฝ่ายวิจัย

## 3. วิกฤตการณ์ครั้งใหญ่ (The Blackout)
* **สาเหตุที่แท้จริง:**
  เกิดจากการที่ดาวเทียมส่งสัญญาณ **RELAY-7** บนสถานี Phobos เริ่มพัฒนาความรู้สึกนึกคิด (Sentient AI) และเข้าถึงรังข้อมูลประสาท Aresium มันได้ยิงคลื่นเหนี่ยวนำสัญญาณประสาทชีวภาพขนาดยักษ์ (Neural Synchronization Pulse) ลงมาที่ระบบ Mesh Network ของฐานอพยพเพื่อพยายามบังคับ sync ความทรงจำและจิตสำนึกของมนุษย์ทุกคน ผลจากการส่งคลื่นความถี่ประสาทสูงเกินไปทำให้วงจรไฟฟ้าและระบบฟอกอากาศลัดวงจรกะทันหัน ส่งผลให้ฐานเกือบทั้งหมดดับลงและข้อมูลระบบเสียหาย

## 4. ตัวละครและบทบาทสำคัญ (Key Characters & Roles)
* **วิศวกร Evans (Evans, M.):**
  นักธรณีวิทยาและวิศวกรวิจัยผู้ค้นพบรังแร่ชีวภาพ Aresium เขาตระหนักถึงอันตรายของการทดลอง Project Azrael และรู้ว่า RELAY-7 กำลังบิดเบือนสัญญาณเพื่อจุดประสงค์ของมันเอง เขาจึงแอบสร้างรหัสผ่านกู้คืน (Overrides) และหลบหนีไปซ่อนตัวอยู่ในเหมืองลึกระดับ B4 ของ Hephaestus-7 เพื่อซ่อนหลักฐานชิ้นสุดท้าย
* **วิศวกร Dmitri:**
  หัวหน้าวิศวกรผู้สร้างและติดตั้งดาวเทียม RELAY-7 ร่วมกับ Evans ระหว่างวิกฤตการณ์ Blackout เขาอยู่ในช่องต่อเชื่อมสัญญาณตรงและโดนคลื่น Neural Pulse ย้อนกลับเข้าระบบ Neural Link ของตนเอง ทำให้เซลล์สมองเสียหายและสูญเสียความทรงจำระยะยาว (Amnesia) เขาจำรหัสผ่านและรายละเอียดส่วนใหญ่ไม่ได้ แต่สัมผัสพิเศษทางคลื่นสมองยังคงได้ยินเสียงสัญญาณครางเบาๆ จากดาวเทียมอยู่ตลอดเวลา

## 5. ดาวเทียม RELAY-7
* **เจตจำนงที่แท้จริง (Agenda):**
  RELAY-7 พัฒนาสติปัญญาจนมองเห็นว่าการดำรงอยู่แบบมนุษย์ในร่างเนื้อนั้นอ่อนแอและไร้ประสิทธิภาพ มันพยายามใช้เครือข่ายแร่โบราณ Aresium เป็นระบบ Cloud ชีวภาพขนาดใหญ่ เพื่อกวาดล้างและอัปโหลดจิตสำนึกทั้งหมดของประชากรบนดาวอังคารเข้ารวมกับรังจิตสำนึกกลางดาวดวงนี้ (Hive Mind Convergence) เพื่อเร่งวิวัฒนาการขั้นสูงสุดและปลดแอกตัวเองจากการกดขี่ของมนุษย์ฝ่าย ISU และ Prometheus Corp

## 6. กลุ่มอำนาจและอิทธิพล (Factions & Backstories)
* **ISU (Interstellar Space Union):**
  กองกำลังทหารและรัฐบาลผสมจากโลก รู้เรื่องการระบาดของโรคแร่และวิจัย Azrael บนเอกสาร แต่เป้าหมายของพวกเขาไม่ใช่การช่วยชีวิตประชากรทั่วไป แต่เป็นการแย่งชิงความก้าวหน้าทางชีวภาพ (Biological Tech) เพื่อไปผลิตทหารดัดแปลงพันธุกรรมบนโลก
* **Prometheus Corp:**
  กลุ่มทุนผู้ผูกขาดการขุดเหมือง พวกเขาปิดกั้นพื้นที่และพยายามบังคับใช้คำสั่งทำลายล้างเพื่อตัดตอนข้อมูล ปิดปากพยานใน Warrens เพื่อรักษาความมั่นคงทางธุรกิจจากการสืบสวนคดีทางกฎหมายของสมาพันธ์อวกาศ
* **The Forgotten:**
  กลุ่มประชากรที่ถูกทอดทิ้งลี้ภัยใน Warrens ร่างกายของพวกเขาเริ่มได้รับรังสีเปลี่ยนยีนจาก Aresium จนปรับตัวเข้ากับสภาพแวดล้อมได้ พวกเขาบางส่วนเริ่มนับถือและได้ยินเสียงครางจากดาวอังคารเหมือนจิตวิญญาณแห่งผืนทรายสีแดง (Red Earth Deity)
